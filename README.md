### Description:

This game is an attempt at recreating the famous "Game of Life" simulation thought of by ___. I wanted to build this project, both as a proof of concept and an attempt at building by own assembly and systems knowledge.

Feel free to use any of the code to your own discresion :)

If you have any questions to why I have written something the way I have or have found a bug please reach out!

---
### Make Directions:
In order to build the ROM for the game install the following:\
``wla-dx`` (Assembler/Linker) [``wla-65816`` is the specific assembler and ``wlalinker`` is the linker]

Then when dependencies are installed, run ``$make`` and ``GameOfLife.sfc`` will be made
